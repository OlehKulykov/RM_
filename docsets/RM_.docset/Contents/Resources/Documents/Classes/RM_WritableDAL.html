RM_WritableDAL

open class RM_WritableDAL<InternalType, PublicType>: RM_ReadableDAL<InternalType, PublicType>

Generic implementation of the mutable DAL common functionality. Inherits all readable/immutable functionality and extends with editing functionality.

Note

This functionality should be provided with InternalType for both InternalType and PublicType types.

Warning

For internal purpose only. Use RM_MutableDAL instead.
