  • addBorders(_:color:thickness:)

    Add borders to the view. For each required rectangle edge using appropriate methods, e.g. ‘addTopBorder’, ‘addLeftBorder’, etc.

    Note

    The adding borders sequence is: top, left, bottom, right.

    Example:

    // Add top and left border with the same color and thickness
let borders = view.addBorders([.Top, .Left], color: UIColor.redColor(), thickness: 1.5)

// Add single bottom border
view.addBorders(.Bottom, color: UIColor.redColor(), thickness: 0.5)

    Declaration

    Swift

    public func addBorders(_ edges: UIRectEdge, color: UIColor, thickness: CGFloat) -> [UIView]

    Parameters

    edges

    View edges to that add borders.
    color

    Color of the added borders.
    thickness

    The thickness of the border views, e.g. border width in a case of vertical or height in case of horizontal.

    Return Value

    Array with added borders.

  • setShadow(_:color:radius:opacity:)

    Set shadow for some or all view sides.

    Warning

    Set shadow only after view layout was changed, due to correct view/layer bounds. See example.

    Example: 

    override func layoutSubviews() {
    super.layoutSubviews()
    setShadow(UIEdgeInsetsMake(1, 2.5, 6, 2.5), color: UIColor.blackColor(), radius: 4, opacity: 0.16)
}

    color

    opacity
