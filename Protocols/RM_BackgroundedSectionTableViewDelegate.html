RM_BackgroundedSectionTableViewDelegate

public protocol RM_BackgroundedSectionTableViewDelegate

RM_BackgroundedSectionTableView table view delegate. RM_BackgroundedSectionTableView have no special property/variable for this type, so, extend your UITableViewDelegate with this protocol.

  • tableView(_:sectionBackground:)

    Ask gelegate object to create section background view for the section by it’s index. By default no section background, e.g. nil.

    Declaration

    Swift

    @objc optional func tableView(_ tableView: RM_BackgroundedSectionTableView, sectionBackground section: Int) -> RM_TableViewSectionBackground?

    Parameters

    tableView

    The table view that required for the section view.
    section

    Requested section index.

    Return Value

    Initialized section background view or nil if no background needed.

  • tableView(_:sectionBackgroundEdgeInsets:)

    Ask gelegate object for the section background edge insets. Provide positive values to decrease section background size from top, left, etc. Default - no insets.

    Declaration

    Swift

    @objc optional func tableView(_ tableView: RM_BackgroundedSectionTableView, sectionBackgroundEdgeInsets section: Int) -> UIEdgeInsets

    Parameters

    tableView

    The table view that required for the section insets.
    section

    Requested section index.

    Return Value

    Section background edge insets.

  • tableView(_:sectionBackgroundUnderFooter:)

    Ask gelegate object that section background should be placed also under section footer. Default is false - ignore section footer.

    Declaration

    Swift

    @objc optional func tableView(_ tableView: RM_BackgroundedSectionTableView, sectionBackgroundUnderFooter section: Int) -> Bool

    Parameters

    tableView

    The table view that could place section background under footer.
    section

    Requested section index.

    Return Value

    true - section background also locates under footer, false - ignore section footer.

  • tableView(_:sectionBackgroundUnderHeader:)

    Ask gelegate object that section background should be placed also under section header. Default is false - ignore section header.
