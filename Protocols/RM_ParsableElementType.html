RM_ParsableElementType

public protocol RM_ParsableElementType

Parsable element object protocol which could be initialized with data(JSON, XML, etc) and casted to an internal, primitive values. Like dictionaries, arrays, numbers etc.

  • init(data:)

    Initialize element from data.

    Declaration

    Swift

    init?(data: Data)

    Parameters

    data

    The data object to parse.

    Return Value

    Initialized element or nil if data is not in a supported format for the implementation.

    Key-based subscription to get element by it’s key in a case of dictionary element type.

    Declaration

    Swift

    subscript(key: String) -> Self { get }

    Parameters

    key

    Object string key.

    Return Value

    Element for key or Null if no such value for key present or element is not dictionary.
